# Features:
- Autenticação de usuários
- Formulários
    - Visualização de formulários
    - Registro de respostas
- Dashboards de admin
- Serviços de admin
    - Importação de dados (JSON fornecido)
    - Gerenciamento de formulários (CRUD)
    - Envio de formulários
    - Resultados
        - Gráficos
        - CSV
